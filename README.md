# 🌍 Travel Destination Web Page

## 📌 Project Overview

This project is a simple and informative **Travel Destination Web Page** built using HTML.
It provides details about both **National** and **International** travel destinations.

The webpage helps users explore travel information like best season, budget, attractions, and itinerary planning.

---

## ✨ Features

### 🗺 Destination Types

* National Trips
* International Trips

### 📊 Quick Facts Section

Each destination includes:

* Best Season to Visit
* Recommended Duration
* Estimated Budget Range
* Top Attractions
* Local Transport Information

### 🎯 Things To Do

A list of popular activities and experiences in the selected destination.

### 📝 Mini Itinerary

A short day-wise travel plan for better trip planning.
